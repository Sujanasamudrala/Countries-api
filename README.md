# Countries API

A RESTful API built using Node.js, Express.js, and MongoDB that provides complete CRUD functionality for managing country data. The API includes JWT-based authentication, input validation, search by country name, region-based filtering, alphabetical sorting, pagination, and robust error handling.

## Tech Stack
- Node.js
- Express.js
- MongoDB & Mongoose
- JWT Authentication
- Express Validator
- dotenv
- Nodemon

## Features
- CRUD operations
- JWT Authentication & Authorization
- Search countries by name
- Filter by region
- Alphabetical sorting
- Pagination
- Input validation
- Error handling
